什么是 Meta 标签？为什么它们对现代网页不可或缺？

Meta 标签是 HTML 文档头部（ <head> ）中的特殊标签，用于向浏览器、搜索引擎和社交媒体平台提供关于页面的结构化元数据。虽然这些标签在页面上不可见，但它们直接决定了您的网页在数字世界中的“样貌”和“搜索排名”。

SEO 的基石：Title 与 Description

尽管搜索引擎算法在不断进化，但 title 和 meta description 依然是影响点击率（CTR）最直接的因素。一个优秀的描述应该包含核心关键词，并以“解决问题”为导向，引导用户点击。

  • 标题： 推荐在 60 个字符以内。
  • 描述： 推荐在 150-160 个字符以内。

社交媒体时代的 Open Graph 协议

当用户在微信、Facebook 或 Twitter 上分享您的链接时，Open Graph (og:) 标签决定了分享卡片中显示的图片、标题和描述。如果没有配置这些标签，平台会随机抓取页面上的图片，往往导致排版混乱，严重降低分享点击欲望。

解决“重复内容”的 Canonical 标签

如果您网站的同一个页面可以通过多个 URL 访问（例如 /page 和 /page?ref=ads ），搜索引擎可能会认为这是内容重复。通过设置 rel="canonical" ，您可以告诉搜索引擎哪一个是主版本，从而集中所有链接权重，提升搜索排名。

Meta 标签与安全性

除了 SEO，某些 Meta 标签还具有安全防护功能。例如：

  • robots: 告诉爬虫不要索引当前页面或不要追踪页面上的链接。
  • referrer: 控制浏览器发送 Referer 信息的策略。
