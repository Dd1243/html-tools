什么是 IP 地址？

基础概念

IP 地址是分配给网络设备的数字标签。IPv4 由 32 位二进制组成，通常以“点分十进制”表示（如 192.168.1.1）。

转换意义

将 IP 转换为十进制或十六进制有助于数据库存储优化（节省空间、索引快）、网络编程（Socket 接口要求）以及安全分析（规避简单过滤）。

IPv4 地址的格式与组成

虽然计算机内部使用二进制处理，但点分十进制法更符合人类阅读习惯。每个部分（Octet）的值范围是 0 到 255。

计算示例: 192.168.1.1 = 11000000.10101000.00000001.00000001 (二进制)

公网 IP 与私网 IP 的区别

RFC 1918 定义了三组私有地址范围，专门用于局域网内部：

  • 10.0.0.0 - 10.255.255.255 (A 类)
  • 172.16.0.0 - 172.31.255.255 (B 类)
  • 192.168.0.0 - 192.168.255.255 (C 类)

这些私有地址通过 NAT 技术与公网通信，有效地缓解了 IPv4 地址枯竭的问题。
