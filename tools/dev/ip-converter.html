什么是 IP 地址？

IP 地址（Internet Protocol Address）是分配给连接到网络的每个设备的一个数字标签。它主要有两个功能：标识主机（或网络接口）和提供设备在网络中的位置。最常用的 IP 地址版本是 IPv4，它由 32 位二进制数组成。

IPv4 地址的格式与组成

虽然计算机内部使用二进制处理 IP 地址，但为了人类可读，我们通常使用“点分十进制”表示法。例如 192.168.1.1。这个地址实际上由 4 个 8 位（Octet）的部分组成，每个部分的值范围是 0 到 255。

计算过程如下：
192.168.1.1 = 11000000.10101000.00000001.00000001 (二进制)

为什么需要将 IP 地址转换为十进制或十六进制？

  • 数据库存储优化： 在数据库中存储一个 4 字节的整数比存储一个 15 字节的字符串更节省空间，且索引查询速度更快。
  • 网络编程： 许多底层的套接字（Socket）编程接口要求使用整数形式的 IP 地址。
  • 安全分析： 在分析恶意代码或混淆的 URL 时，攻击者有时会使用十进制或十六进制形式的 IP 来规避简单的字符串过滤。
  • 数学计算： 判定一个 IP 是否属于某个子网掩码（CIDR）范围时，将其转换为整数进行位运算是最直接的方法。

公网 IP 与私网 IP 的区别

并不是所有的 IP 地址都能在互联网上公开访问。RFC 1918 定义了三组私有地址范围，专门用于局域网内部：

  • 10.0.0.0 - 10.255.255.255 (A 类)
  • 172.16.0.0 - 172.31.255.255 (B 类)
  • 192.168.0.0 - 192.168.255.255 (C 类)

这些私有地址通过 NAT（网络地址转换）技术与公网通信，有效地缓解了 IPv4 地址枯竭的问题。
