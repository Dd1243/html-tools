什么是 Favicon 网站图标？

Favicon 是 "Favorite Icon" 的缩写，也称为网站图标、快捷方式图标或浏览器图标。它通常显示在浏览器的标签页、收藏夹、地址栏以及移动设备的桌面上。

为什么需要 Favicon 检测器？

  • 开发者调试： 检查自己的网站是否正确配置了各种尺寸的图标（如 Apple Touch Icon 或 PWA 图标）。
  • 品牌分析： 提取并研究其他优秀网站的品牌标识设计。
  • 资源获取： 快速获取某个网站的高清图标用于收藏、导航页制作或 UI 设计参考。

本工具支持检测的图标格式

我们的探测算法会覆盖以下常见的图标路径：

  • 标准格式： /favicon.ico (32x32, 16x16)
  • 现代格式： /favicon.png , /favicon.svg
  • 移动设备： apple-touch-icon.png (iPhone/iPad 专用)
  • PWA/Android： /icon-192x192.png , /icon-512x512.png
  • Windows 磁贴： /mstile-150x150.png
