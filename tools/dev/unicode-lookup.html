什么是 Unicode？为什么我们需要查询它？

Unicode（万国码）是计算机科学领域里的一项业界标准，旨在为世界上超过 100 万个字符提供统一且唯一的数字编号。无论是中文、英文、表情符号（Emoji），还是古老的象形文字，在 Unicode 标准中都有其对应的 码点（Code Point） 。

Unicode 查询工具 可以帮助开发者解决以下具体问题：

  • 乱码排查： 当网页出现 锟斤拷 或方框时，通过查询原始字符的编码，可以定位是数据库字符集还是前端渲染的问题。
  • 特殊字符调用： 在编写 CSS 或 JavaScript 时，经常需要使用 \uXXXX 格式。本工具能直接提供这些格式的代码。
  • Emoji 深度研究： 每一个 Emoji 其实都是由一个或多个 Unicode 码点组成的，通过本工具可以拆解它们的构成。

Unicode 常见知识点

1. 什么是码点 (Code Point)？

码点是 Unicode 分配给每个字符的唯一数字，通常以 U+ 开头后跟十六进制数字。例如，汉字“一”的码点是 U+4E00 。

2. Unicode 与 UTF-8 的区别

Unicode 是 字符集 （定义了字符与数字的对应关系），而 UTF-8 是一种 编码方案 （定义了如何将这些数字转换为计算机存储的字节流）。一个 Unicode 字符在 UTF-8 编码下可能占用 1 到 4 个字节。

3. 在代码中使用 Unicode

在 HTML 中，您可以使用 中 渲染字符；在 CSS 中使用 content: "\4E2D"; ；在 JavaScript 中使用 "\u4E2D" 。本工具为您一键准备了这些代码。
