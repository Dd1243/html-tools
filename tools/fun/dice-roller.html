📖 骰子模拟器：不仅仅是随机数

骰子是人类文明中最古老的决策工具之一。本工具采用 Web Crypto API 生成密码学安全的随机数（CSPRNG），确保每一次投掷在物理与数学意义上都是绝对公平的。

1. 为什么我们需要多种骰子？

在 D&D (地下城与龙) 或其他 TTRPG 游戏中，不同面数的骰子代表了不同的可能性分布。

  • D6 (六面骰)： 最经典的骰子，代表平稳的概率分布。
  • D20 (二十面骰)： 命运之骰，用于决定攻击是否命中、检定是否成功。
  • D100 (百分骰)： 用于决定随机事件发生的百分比概率。

2. 期望值与概率分布

当你投掷 2D6 时，最可能出现的结果是 7。这是因为 7 的组合最多（1+6, 2+5, 3+4 ...）。随着骰子数量增加，结果会越来越接近正态分布。本工具内置了统计功能，助你分析投掷趋势。

3. 加密级随机性 (CSPRNG)

普通的 Math.random() 在某些高频率场景下可能存在周期性偏差。我们使用浏览器原生的加密接口，每秒可生成数千个高质量随机种子，确保你的“自然 20”是真正的运气使然。
