EXIF 隐私保护完整指南

什么是 EXIF 信息？

EXIF（Exchangeable Image File Format，可交换图像文件格式）是数码相机和手机在拍摄照片时自动嵌入的元数据。这些数据对你不可见，但包含大量隐私信息。

EXIF 可能包含哪些信息？

  • GPS 坐标 ：拍摄地点的精确经纬度（误差可能在几米内）
  • 拍摄时间 ：照片拍摄的精确日期和时间
  • 设备信息 ：手机/相机品牌、型号（如 iPhone 15 Pro）
  • 拍摄参数 ：光圈、快门速度、ISO 感光度
  • 软件信息 ：处理图片所用的软件版本
  • 方向信息 ：拍摄时设备的朝向

为什么需要移除 EXIF？

位置隐私： 在社交媒体分享照片时，GPS 信息可能暴露您的家庭地址、工作地点或常去场所。犯罪分子可能利用这些信息实施跟踪或入室盗窃。

时间隐私： 拍摄时间戳可能暴露您的日常作息规律，让他人知道您何时在家、何时外出。

设备隐私： 设备型号信息可能让不法分子了解您使用的设备价值，成为盗窃目标。

本工具如何移除 EXIF？

本工具通过 HTML5 Canvas 技术重绘图片：将原图绘制到 Canvas 画布上，再从 Canvas 导出为新图片。这个过程会生成一张全新的图片文件，原始 EXIF 数据不会被复制到新文件中，从而彻底清除所有元数据。

这种方法与简单删除 EXIF 标签不同，它是通过像素级重绘实现的，更加彻底可靠。

哪些平台会保留 EXIF？

  • 会保留： 直接文件分享、邮件附件、部分云盘、论坛直接上传
  • 会移除： 微博、Instagram、Twitter/X、微信朋友圈（压缩时）
  • 不确定： 各平台政策随时可能变化，保险起见建议上传前自行移除

常见问题解答

Q：图片质量会损失吗？
重绘过程会有轻微质量损失（尤其 JPEG）。可通过调节质量滑块控制，92% 对普通使用几乎无感知差异。

Q：PNG 图片有 EXIF 吗？
PNG 格式本身不支持标准 EXIF，但可能含有 PNG 元数据块（如 tEXt、iTXt）。重绘同样会清除这些信息。

Q：图片文件会上传吗？
完全不会。所有处理使用浏览器的 Canvas API 在本地完成，图片数据不离开您的设备。
