为什么用这个工具？

学习摩尔斯、解密电码或做电台练习时，需要快速互转并核对结果。本工具本地运行，支持双向转换、示例填充和音频播放。

双向互转 文本转电码、电码转文本，实时同步，字符与单词分隔清晰。
音频播放 一键播放点划声，方便听写练习与节奏记忆。
手机友好 双栏自动变单栏，大按钮触控，移动端也能顺手操作。

使用提示

  • 字母之间用空格，单词之间用 /
  • 支持 A–Z、0–9 及常见标点
  • 内容仅在本地转换，不会上传服务器
