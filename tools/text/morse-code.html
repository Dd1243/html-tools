什么是摩尔斯电码？

摩尔斯电码（Morse Code）是一种时断时续的信号代码，通过不同的排列顺序来表达不同的英文字母、数字和标点符号。它由塞缪尔·摩尔斯（Samuel Morse）在 1830 年代发明，最初用于电报机进行远距离通信。电码由两种基本单位组成：短信号“滴”（Dot，用 . 表示）和长信号“哒”（Dash，用 - 表示）。

摩尔斯电码的编码规则

为了确保信息的准确传递，摩尔斯电码遵循严格的时间比例规则：

  • 点（Dot）： 基础单位，长度为 1t。
  • 划（Dash）： 长度等于 3 个点，即 3t。
  • 字符内间隙： 同一个字母内的点或划之间，间隔为 1t。
  • 字符间间隙： 字母与字母之间，间隔为 3t（在文本表示中常用空格）。
  • 词间间隙： 单词与单词之间，间隔为 7t（在文本表示中常用 / 或三个空格）。

问：为什么我听不到播放的声音？

问：电码中的“/”代表什么？
