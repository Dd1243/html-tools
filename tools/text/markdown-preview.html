什么是 Markdown 预览器？它能解决什么问题？

虽然 Markdown 的语法非常直观，但在处理复杂的表格、嵌套列表或多级标题时，仅凭纯文本源码很难直观地判断排版效果。 Markdown 在线预览器 正是为了填补这一空白而生。它能将你的 MD 源码实时转换为浏览器可识别的 HTML 格式，帮助你在正式发布文章（如发布到 GitHub README、个人博客或技术社区）前，进行最后的视觉确认。

1. 为什么它是创作者的必备工具？

  • 消除排版焦虑： 即时看到标题级别、列表缩进和加粗效果，避免发布后出现排版错误。
  • 支持复杂语法： 支持 GitHub Flavored Markdown (GFM)，包括任务列表、表格和自动链接。
  • 跨平台兼容性： 我们的预览样式接近大多数主流平台的默认渲染效果，确保所见即所得。

2. 技术核心：性能与安全

我们的预览器集成了业界领先的 Marked.js 解析引擎，即使处理上万行的超长文档，也能保持毫秒级的实时响应。在安全性方面，我们引入了 DOMPurify 进行实时过滤，防止任何潜在的跨站脚本（XSS）注入，确保您的创作环境绝对安全。

3. 常见问题 (FAQ)

问：该工具会将我的文章上传到服务器吗？
答：绝对不会。所有渲染逻辑均在您的本地浏览器中完成。您的内容永远留在您的计算机上。

问：支持数学公式和流程图吗？
答：目前专注于标准 GFM 语法的极致性能预览。对于更高级的插件功能，我们正在持续评估中。
