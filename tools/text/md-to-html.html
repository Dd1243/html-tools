为什么用这个工具？

写 README、技术文档或博客草稿时，常需要把 Markdown 转成 HTML。本工具基于 GFM 语法本地转换，支持实时预览与源码导出，无需上传内容。

实时预览 输入即渲染，标题、列表、代码块、表格即时可见。
源码导出 一键复制 HTML，或下载完整 HTML 文件。
手机友好 双栏自动变单栏，触控按钮更大，移动端也能顺手编辑。

支持的常见语法

  • 标题、段落、加粗、斜体、删除线
  • 有序/无序列表、任务列表
  • 链接、图片、引用、表格
  • 行内代码与代码块（GFM）
