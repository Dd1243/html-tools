会议议程生成器

规划会议结构，掌控讨论节奏。一键生成专业的议程文档。

📅 会议基本信息
⚡ 快速模板
📝 议程项目清单
会议总时长: 0 分钟
👁️ 预览与导出

怎么用

  • 标题写目标，不写「例会」空泛词。
  • 议题 3–7 个，决策项靠前，同步项靠后。
  • 填主持人与参会，便于会前预读责任。
  • 生成 Markdown 贴进文档库，或 HTML 发邮件。
  • 会中按总分钟控场；超时砍掉低优先级议题。

常见问题

Q: 输出为空？
A: 先点对应生成按钮再复制。
Q: 总分钟不准？
A: 检查每条议题时长字段是否填数字。
Q: 模板会覆盖吗？
A: 加载模板可能替换议题结构，先保存/导出再试。
Q: 和会议成本计算器？
A: 议程控结构；成本器显金钱，可同时投屏。

会议议程生成器（团队工具版）完整指南

编排会议元信息与议题列表：标题 #meetTitle、日期 #meetDate、时间 #meetTime、地点 #meetLoc、主持人 #meetHost、参会 #meetAtt。议题列表可增删改，显示总分钟 #totalMin。输出区 #output 支持生成 Markdown/纯文本/HTML（genMD/genTxt/genHTML），可复制、下载、加载模板、分享状态。键 meet-agenda-data。与办公类「议程制作」页可能并存但数据不互通，以本页控件为准。

功能对照

  • addItem/updateIt/delItem/renderItems 维护议题。
  • updateUI 汇总时长等。
  • loadTpl 快速套常见会议结构。
  • saveState/loadState；分享用 Base64 等编码本地快照到 URL。
  • copyResult/downloadResult 分发议程。

高效会议编排

  1. 标题写目标，不写「例会」空泛词。
  2. 议题 3–7 个，决策项靠前，同步项靠后。
  3. 填主持人与参会，便于会前预读责任。
  4. 生成 Markdown 贴进文档库，或 HTML 发邮件。
  5. 会中按总分钟控场；超时砍掉低优先级议题。

边界

  • 不自动发日历 ICS（除非另有按钮，以页面为准）。
  • 无冲突检测与视频会议深度集成。
  • 分享链接含会议内容，注意机密。
  • 与 office/agenda-maker 数据键不同，勿假设同步。

本地与分享

键 meet-agenda-data 存本机；分享适合远程对齐同一版议程。公共电脑用完清空。系列例会可改日期复用议题骨架。主题切换不影响正文。

输出格式怎么选

Markdown 进文档库与 Git；纯文本进 IM；HTML 进邮件富文本。生成后再 copy/download，避免复制旧缓存。模板适合站会/评审骨架，加载后按本次目标删改。总分钟是各议题之和，可加「缓冲」议题。

与 office 议程页

站内可能另有 agenda-maker，存储键与导出能力不同，选一处维护避免双份。本页键 meet-agenda-data。分享链接对齐远程参会者。机密并购会勿用公开电脑。

本地数据与使用边界

本页草稿与结果主要保存在当前浏览器环境，清除站点数据、更换设备或隐私模式可能导致丢失，重要报价、名片、议程、投票与表单请及时导出、复制或打印备份。公共电脑使用完毕后应清空敏感信息。工具用于提升效率，不替代正式商务、项目管理或合规流程，请以组织制度与法规为准。

生成 HTML/MD 时注意

输出区的 HTML 字符串里可能含标题标签用于导出文档结构，那是生成结果不是页面第二个主标题。复制前确认已 gen 最新版。Markdown 适合知识库；TXT 适合IM；下载便于归档。模板加载会改议题，先分享或下载备份。

控时与成本

总分钟 totalMin 约束议程；可与会议成本计算器同屏。参会人名单写全便于会前预读。键 meet-agenda-data 本机存储；分享 URL 含快照注意机密。与 office/agenda-maker 不互通数据。

备份与公共环境注意

重要结果请下载、复制或打印留存，勿只依赖浏览器缓存。公共电脑与共享账号使用后清空页面数据与剪贴板。本工具提升个人与小团队效率，不替代正式商务系统、印刷工艺或带身份认证的投票选举流程。
