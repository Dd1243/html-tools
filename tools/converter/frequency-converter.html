深入理解频率单位转换

频率是描述周期性现象在单位时间内发生次数的量。在日常生活中，我们通过赫兹了解声音的音调、通过兆赫了解无线电广播的频道、通过吉赫了解处理器的运行速度。准确的频率转换是现代通信和工业控制的基础。

常见的频率单位解析

  • 赫兹 (Hertz, Hz)： 国际单位制 (SI) 中的频率单位。1 赫兹代表每秒发生 1 次周期性事件。它以物理学家海因里希·赫兹的名字命名。
  • 兆赫 (MHz) 与 吉赫 (GHz)： 常用于电子通信。WiFi 信号通常在 2.4 GHz 或 5 GHz 频段运行，而调频广播 (FM) 则在 88-108 MHz 范围内。
  • 转每分 (RPM)： 机械领域常用的转速单位。它描述物体每分钟旋转的圈数。在工业电机和汽车引擎规格中非常常见。
  • 弧度每秒 (rad/s)： 物理学中的角速度单位，描述物体每秒转过的弧度。

核心换算公式

频率单位之间的换算通常遵循简单的比例关系：

转换方向 公式 说明
RPM ➔ Hz Hz = RPM / 60 每秒转数 = 每分转数除以 60
Hz ➔ rad/s rad/s = Hz * 2π 一圈等于 2π 弧度
kHz ➔ MHz MHz = kHz / 1000 标准十进制前缀换算

频率与波长的关系

在波动物理学中（如光波或电磁波），频率与波长成反比关系。公式为： v = f * λ （其中 v 是波速，f 是频率，λ 是波长）。对于光速而言，频率越高，波长就越短。这也是为什么高频 5G 信号虽然传输速度快，但穿透力和覆盖范围相对较小的原因。
