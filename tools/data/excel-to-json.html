什么是 Excel/CSV 转 JSON？

在现代 Web 开发中，JSON（JavaScript Object Notation）是数据交换的标准格式。然而，大多数原始数据（如业务报表、配置清单）通常存储在 Excel 或 CSV（逗号分隔值）表格中。本工具的作用就是作为中间件，将人类易读的表格结构一键转化为程序易读的 JSON 结构。

为什么选择本工具？

  • 智能识别：无需手动指定分隔符。我们内置的算法会自动识别您是从 Excel 粘贴的（制表符分隔）还是标准的 CSV 文件。
  • 多种结构支持：支持将表格转换为对象数组（适合 API 响应）、二维数组（适合数据分析）或简单的键值对。
  • 本地运行，安全无忧：您的数据完全在浏览器本地处理，绝不会被上传到服务器。处理数万行数据也能在瞬间完成。
  • 格式美化：输出的 JSON 经过自动缩进处理，方便直接阅读和代码集成。

使用小贴士

如果您是从 Excel 粘贴数据，请确保选中了包含标题行的完整区域。如果表格中包含特殊字符（如换行符或引号），我们的解析器遵循 RFC 4180 标准，能够完美处理带引号的复杂字段。
