什么是 JSON 反扁平化？

JSON 反扁平化（Unflattening）是 JSON 扁平化的逆向过程。它能将一个具有“路径化键名”的扁平对象（如使用点号连接的 address.city）重新映射为深层嵌套的对象结构。在数据处理流程中，我们经常为了存储或表格展示而将 JSON 展平，而当数据需要导回应用系统时，反扁平化就是恢复其业务逻辑结构的关键步骤。

核心技术特性

  • 路径递归解析：我们的引擎可以处理无限深度的路径层级，精准重构对象原型。
  • 智能数组识别：如果开启此功能，路径中的纯数字片段（如 items.0 或 items[0]）将被自动识别为数组索引，而不是普通的属性键。
  • 自定义分隔符：不仅支持标准的点号（.），您还可以根据特定系统的导规范使用斜杠（/）、下划线（_）等作为分隔符。

隐私与本地计算

与 WebUtils 的其他工具一样，JSON 反扁平化过程完全在浏览器本地内存中完成。您的敏感数据绝不会离开您的计算机，不产生任何网络请求，真正做到了安全与极速的统一。
