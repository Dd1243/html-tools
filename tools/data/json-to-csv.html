JSON 转 CSV 转换器

递归展平复杂结构，轻松导入电子表格

为什么需要将 JSON 转换为 CSV？

JSON (JavaScript Object Notation) 是现代 Web API 的标准数据格式，但它并不适合非技术人员查看或进行深度的数据透视分析。CSV (Comma-Separated Values) 则几乎兼容所有的电子表格软件（如 Microsoft Excel、Google Sheets）。将 JSON 转换为 CSV，可以让业务分析师、市场人员更方便地对数据进行筛选、排序和绘图。

本工具的特色：深度展平 (Flattening)

普通转换器往往只能处理简单的扁平 JSON 数组。而本工具支持递归展平。如果你的 JSON 数据中包含嵌套对象（例如 {"user": {"name": "Alice"}}），它会自动将其转换为带有点号分隔符的列（例如 user.name）。这确保了即便数据结构再复杂，也能在 CSV 中完整保留信息。

安全性说明

您的隐私至关重要。本站所有的 JSON 转换逻辑均在您的浏览器本地运行。数据不会被上传到任何服务器，这不仅保证了极高的转换速度，也杜绝了敏感数据泄露的风险。即使是处理数兆大小的文件，也能在瞬间完成。

如何导出 Excel 不乱码？

如果您发现导出的 CSV 在 Excel 中打开出现乱码，请确保在下载时选择了带有 BOM 的 UTF-8 编码。我们的下载功能已默认处理了这一问题，确保中文内容在 Excel 中能被正确识别和展示。
