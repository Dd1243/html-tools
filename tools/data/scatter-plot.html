什么是散点图及其在数据分析中的作用？

散点图（Scatter Plot）是统计学中用于展示两个连续变量之间关系的最直观工具。通过在直角坐标系上绘制点，每个点的坐标（x, y）代表一个观测值的两个属性。散点图的主要目的是识别变量之间是否存在相关性、相关性的强度以及关联的方向（正相关、负相关或无相关）。

如何解读相关系数 (R)？

在我们的工具中，会自动为您计算皮尔逊相关系数（Pearson Correlation Coefficient）。R 的取值范围在 -1 到 +1 之间：

  • R = 1: 完美的正相关，所有点都精确落在一条上升的直线上。
  • 0.7 < R < 1: 强正相关，变量随另一变量显著增加。
  • R = 0: 无相关性，变量之间没有任何线性规律。
  • -1 < R < -0.7: 强负相关，变量随另一变量显著减少。

线性回归方程的意义

本工具不仅绘制点，还会自动拟合一条“最佳拟合线”（线性回归线）。回归方程 y = mx + b 揭示了变量间的数学模型。通过这个模型，你可以预测当自变量 X 变化时，因变量 Y 的大致趋向。这在商业预测、工程实验和经济分析中极具价值。
