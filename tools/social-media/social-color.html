设计规范：为什么官方品牌色（Brand Colors）至关重要？

在进行网页设计、APP 开发或社交媒体物料制作时，准确使用社交平台的官方配色不仅仅是为了美观，更是品牌专业度的体现。每一个主流平台都有严格的视觉识别系统（VI），使用错误的颜色可能会导致用户产生违和感，甚至违反平台的品牌使用指南。

品牌色查询器能解决哪些具体问题？

  • 确保设计一致性： 当你在网页底部添加“关注我们”的图标时，使用官方的“Twitter 蓝”或“微信绿”能让图标看起来更原生、更可信。
  • 提升 UI 交互体验： 在集成社交登录（OAuth）功能时，登录按钮的颜色若符合用户心理预期，能显著提升点击转化率。
  • 节省查找时间： 无需翻阅各大平台晦涩难懂的开发者文档，本工具为您汇总了全量、最新的颜色代码。

主流平台配色深度解析

1. 小红书 (Xiaohongshu)： 标志性的“小红书红”（#FF2442）充满了活力与种草欲望。在设计相关海报时，配合纯白背景和大字报风格，能瞬间捕捉受众注意力。

2. Instagram： 它的品牌色不仅是一个单色，而是一个复杂的渐变色（紫-红-黄）。本工具提供了渐变的关键色值，帮助您在 CSS 中完美复刻。

3. TikTok： 独特的电子青（#25F4EE）与震动红（#FE2C55）在黑色背景下产生的“色差重影”效果，是其品牌年轻化、潮流化的核心视觉资产。

4. LinkedIn： 沉稳的“领英蓝”（#0A66C2）传达了专业、信任与商务属性。它是 B2B 营销物料中的核心色调。

如何将这些颜色应用到代码中？

CSS 变量： 建议在项目的 :root 中定义社交品牌色变量，例如：
--color-wechat: #07C160;
这样在后续维护或平台颜色微调时，只需修改一处即可全局生效。

专家建议：无障碍与对比度

虽然官方品牌色很漂亮，但在实际应用中（尤其是文字背景色），请务必检查对比度。例如，在淡绿色的微信背景上使用白色文字可能不符合 WCAG 无障碍标准。本工具提供的颜色主要用于图标、边框和按钮背景，请结合实际场景灵活调整亮度。
