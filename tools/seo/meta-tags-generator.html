常见问题

Meta 标签生成器能解决什么？

每个重要页面都需要清晰的 title、description，以及面向社交平台的 Open Graph / Twitter 标签。手动拼写容易漏字段、长度失控或重复输出。本工具把常用字段表单化，实时生成可复制的 HTML 片段，并提供搜索结果与社交卡片预览。

必填与建议字段

  • title：页面主标题，品牌名可放后段
  • description：120–160 字中文更稳妥，写清受众与价值
  • canonical / og:url：指定规范地址
  • og:image / twitter:image：绝对路径大图
  • robots：按需 index/follow 或 noindex

最佳实践

  1. 一页一 title，避免整站同名
  2. description 与首屏内容一致，拒绝标题党
  3. 模板站注意分页、筛选页的 noindex 策略
  4. 生成后用实机分享测试，并清理平台缓存

常见坑

  • CMS 主题与插件各输出一套 meta，互相覆盖
  • SPA 仅客户端注入，爬虫看不到
  • 图片用相对路径导致社交抓取失败
  • 中英文长度混算导致截断
