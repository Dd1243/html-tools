图片加文字工具

快速在照片上添加说明、日期、水印或创意文案。支持丰富的字体、描边及位置调整，让每一张图都开口说话。

✍️
点击上传 或 拖拽图片至此

怎么用

  • 上传清晰底图，避免先过度压缩。
  • 写短文案（品牌名/价格/日期），少堆长段落。
  • 调颜色与描边保证对比度，再微调偏移。
  • 下载 PNG 保透明与清晰（若导出支持）；再发社交。
  • 敏感图可先去 EXIF 再加字发布。

常见问题

Q: 文字看不见？
A: 检查颜色是否与底图接近、透明度是否过低、偏移是否出画。
Q: 下载模糊？
A: 底图像素不足或导出缩放；用更高清原图。
Q: 能拖拽文字吗？
A: 以偏移数值调节为主（若无拖拽手势）。

图片加文字工具完整使用指南

在本地图片上叠文字后导出。拖拽 #dropZone 或选择文件 #fileInput 加载底图；编辑区 canvas 预览。文案 #textContent，字体 #fontFamily、字号 #fontSize、颜色 #fontColor、不透明度 #opacitySlider；描边色 #strokeColor、描边宽 #strokeWidth；偏移 #offX/#offY 调整位置。handleFile 读入，render 重绘底图与文字。#downloadBtn 导出结果。适合水印、标价、封面字，不是完整平面设计软件，也不自动识别图中最佳留白区域。

参数如何影响画面

  • 字号过大易出画，过小手机端难读；先预览再下载。
  • 描边让浅字在复杂底上可读；线宽 1–3 常够用。
  • 不透明度做半透明水印，避免完全挡住主体。
  • 偏移相对画布定位，改图尺寸后可能需重调。
  • 字体依赖本机/页面可用字体列表。

推荐流程

  1. 上传清晰底图，避免先过度压缩。
  2. 写短文案（品牌名/价格/日期），少堆长段落。
  3. 调颜色与描边保证对比度，再微调偏移。
  4. 下载 PNG 保透明与清晰（若导出支持）；再发社交。
  5. 敏感图可先去 EXIF 再加字发布。

边界

  • 单层文字为主，无多图层文字框系统（以 UI 为准）。
  • 无曲线绕排、无竖排复杂中文禁则引擎。
  • 全本地 canvas，不上传服务器。
  • 与「文字转图片」不同：本页基于已有照片叠字。

场景与注意

电商角标、活动海报字、内部资料水印「内部资料」都适合。水印勿挡住关键信息。公共电脑用完关闭页面。需要圆角或滤镜可串联其他媒体工具后再加字或先处理底图。

水印与营销字实操

品牌水印用半透明+角落偏移；价格字用粗字号+描边。多行文案用换行，避免单行过长出画。改字号后重看预览再下载。导出后可用压缩减体积。本地处理，公共电脑注意底图隐私。
