关于在线音频可视化器

WebUtils 提供的在线音频可视化器是一款强大的纯前端工具，能够将您的音频信号实时转化为精美的视觉动画。无论您是音乐爱好者、播客制作人还是 Web 开发者，都可以通过它直观地观察音频的频率分布和波形变化。

核心功能

  • 多种显示模式： 支持频谱柱状图 (Bars)、波形图 (Wave) 和圆形频谱 (Circle) 等多种可视化效果。
  • 实时交互： 支持本地音频文件 (MP3, WAV, OGG 等) 播放，同时支持麦克风实时输入。
  • 个性化定制： 提供多种颜色主题（渐变、青色、红色、彩虹），满足不同的视觉需求。
  • 高性能渲染： 基于 Web Audio API 和 HTML5 Canvas 开发，确保在现代浏览器中流畅运行。
  • 隐私安全： 所有音频数据仅在本地处理，不会上传到任何服务器。

如何使用

  1. 点击“选择音频文件”上传您本地的音乐。
  2. 或者点击“使用麦克风”授权浏览器访问您的音频输入设备，进行实时可视化。
  3. 在下方下拉菜单中切换不同的“可视化样式”和“颜色主题”。
  4. 使用自带的播放器控制音频的播放、暂停和进度。

常见问题 (FAQ)

Q: 为什么麦克风没有反应？

A: 请确保您已在浏览器弹出框中点击了“允许”访问麦克风。部分浏览器可能需要通过 HTTPS 协议才能启用该功能。

Q: 能够导出可视化视频吗？

A: 目前该工具仅支持实时显示。您可以结合屏幕录制工具来捕捉可视化画面。

Q: 它是免费的吗？

A: 是的，WebUtils 所有的工具都是完全免费且开源的。
