文字转语音工具使用指南

什么是文字转语音(TTS)?

文字转语音（Text-to-Speech，简称TTS）是一种将书面文字转换为语音输出的技术。本工具使用浏览器内置的 Web Speech API，无需安装任何插件或上传文件即可在本地实现语音合成。

主要使用场景

  • 辅助阅读：帮助视觉障碍用户或阅读困难者获取内容
  • 语言学习：听标准发音纠正口语，学习外语
  • 内容创作：快速预听文章或脚本的朗读效果
  • 长文朗读：解放双眼，边听边做其他事情
  • 英语练习：使用英文声音朗读文章，提升听力

如何选择最佳声音?

不同操作系统预装的声音不同：

  • Windows：内置微软 Zira、David 等声音，中文有 Yaoyao、Kangkang
  • macOS：内置 Siri 风格的高质量声音，中文、粤语均支持
  • Chrome：可访问 Google 在线语音，质量更高但需联网

声音旁带 ☁️ 表示在线声音，质量通常更好；★ 表示系统默认声音。

语速、音调、音量说明

  • 语速：范围 0.5x~2x，0.5x 最慢，1.0x 为正常速度，2.0x 最快
  • 音调：范围 0.5~2，低值声音更低沉，高值声音更尖细
  • 音量：0% 静音，100% 为最大音量

常见问题解答

Q: 为什么声音列表是空的?
A: 部分浏览器需要等待几秒加载声音列表。请稍等后重新刷新页面，或切换到 Chrome 浏览器使用。
Q: 朗读会上传我的文字吗?
A: 使用本地声音时完全不上传；使用在线声音（☁️）时，文字会发送到对应语音服务（如 Google），请注意隐私。
Q: 支持多长的文字?
A: Web Speech API 通常支持数千字，但建议每次朗读不超过 5000 字，过长可能出现截断。
Q: 能保存为音频文件吗?
A: 浏览器原生 Web Speech API 暂不支持导出音频文件。如需保存，可使用系统录音功能录制播放内容。
