视频缩略图提取使用指南

什么是视频缩略图提取?

视频缩略图提取是指从视频文件的特定时间点截取静态图片帧的过程。本工具使用 HTML5 Canvas API 和 Video 元素，完全在浏览器本地处理，无需上传视频到服务器，保护您的隐私。

三种批量提取模式

  • 按时间间隔 ：每隔 N 秒提取一帧，适合均匀采样长视频
  • 按数量均分 ：将视频等分为 N 段，各取一帧，适合快速预览
  • 场景变化检测 ：自动识别画面突变点，智能提取关键帧

使用技巧

点击已提取的缩略图可以切换选中状态，选中多张后可以批量下载为 ZIP 文件。点击时间标签可跳转到视频对应时间点。

Q: 支持哪些视频格式?
A: 支持浏览器能播放的所有格式，包括 MP4 (H.264)、WebM、OGV 等，具体取决于您的浏览器。
Q: 视频文件大小有限制吗?
A: 无服务器限制，但受限于浏览器内存，建议单文件不超过 2GB。
Q: 场景检测算法是如何工作的?
A: 工具每 0.5 秒采样一次，将相邻帧缩小至 64×36 像素后计算像素差异值，超过阈值则判定为场景变化。
