关于在线ICO图标查看器

ICO 图标查看器是一个专业的 Web 工具，专门用于查看、解析和提取 ICO（Windows 图标）和 CUR（静态光标）文件。ICO 文件通常包含多个不同分辨率和色深的图标，以便在不同场景下（如桌面、任务栏、浏览器标签页）显示最佳效果。我们的工具可以帮您把这些隐藏的尺寸全部找出来。

🔍 深度多尺寸解析

自动识别 ICO 文件中包含的所有分辨率（如 16x16, 32x32, 48x48, 64x64, 128x128, 256x256 等），不遗漏任何细节。

⚡ 一键转换为 PNG

支持将提取出的图标即时转换为高清 PNG 格式下载，非常适合 UI 设计师和开发者提取网站 Favicon 或软件图标。

🔒 纯本地安全处理

文件解析完全在您的浏览器前端进行，无需上传到任何服务器。您的隐私和素材安全得到 100% 保障，且处理速度极快。

如何使用 ICO 查看器？

1. 上传文件： 点击上传区域或直接将 .ico/.cur 文件拖入页面。

2. 预览图标： 工具会立即列出该文件中包含的所有图标尺寸及其格式（PNG 或 BMP）。

3. 下载或复制： 点击“下载 PNG”将特定尺寸保存到本地，或点击“复制”直接将图片存入剪贴板。

什么是 ICO 文件？

ICO 是 Windows 系统的图标格式。它与普通图片格式（如 JPG/PNG）最大的区别在于它是一个“容器”。一个 .ico 文件内部可以存储多张不同大小的图片。当 Windows 需要显示图标时，它会根据当前显示比例（如缩放 150%）自动从 ICO 文件中挑选最合适的尺寸，从而避免拉伸造成的模糊。
