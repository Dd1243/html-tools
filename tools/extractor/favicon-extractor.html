什么是 Favicon 网站图标？

Favicon 是 "Favorite Icon" 的缩写，是出现在浏览器标签页、地址栏、书签栏以及移动端主屏幕上的小图标。它不仅是网站品牌识别的重要组成部分，还能帮助用户在打开大量标签页时快速定位您的网站。

本工具是如何工作的？

由于浏览器的安全跨域限制（CORS），前端脚本无法直接读取其他网站的源代码。本工具集成了多个业界公认的图标解析服务，包括：

  • Google Favicon API： Google 提供的图标服务，支持多种尺寸（如 16px, 32px, 64px 等）。
  • DuckDuckGo Icons： 快速获取网站默认的 .ico 文件。
  • Apple Touch Icons： 尝试获取适配 iOS 设备的苹果触摸图标，通常分辨率更高。
  • 直接路径解析： 尝试直接访问网站根目录下的标准 favicon.ico 路径。

如何下载提取到的图标？

在图标卡片下方，我们提供了两个按钮：
1. 复制链接： 直接获取该图标的在线 URL，方便在开发中引用。
2. 下载： 将图标下载到您的本地设备。注意：由于浏览器限制，部分图标可能会在新标签页打开，您可以右键点击并选择“另存为”。

服务说明

由于各网站配置不同，提取到的图标分辨率可能有所差异。Google 服务通常提供更清晰的缩放版本。

提取服务节点：

  • DuckDuckGo API
  • Direct Path
