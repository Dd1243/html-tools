为什么要关注紫外线指数？

紫外线辐射是导致皮肤过早老化、皱纹甚至皮肤癌的主要原因。**紫外线指数（UV Index）**是由世界卫生组织（WHO）等国际机构制定的，旨在提醒公众在不同强度的紫外线暴露下应采取何种防护措施。

多维度防护建议

综合考虑了化学防晒（防晒霜 SPF）与物理防晒（遮阳帽、墨镜、阴凉处）的结合。

科学的指数划分

严格遵循国际气象组织标准，将 UV 指数划分为低、中、高、很高、极高五个科学等级。

如何选择合适的防晒策略？

即使在阴天，多达 80% 的紫外线仍能穿透云层。建议养成查看 UV 指数的习惯。对于中等（3-5）及以上强度，涂抹广谱防晒霜并每 2 小时补涂一次是基础；在极高（11+）强度下，最安全的策略是避开正午时段的户外活动。
