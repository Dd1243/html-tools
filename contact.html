联系我们

如果您发现工具结果异常、页面无法访问、内容有误，或希望提出新工具建议，可以通过下面的渠道联系 WebUtils。

反馈渠道

  • GitHub Issues：提交问题或建议
  • GitHub 仓库：查看项目源码

可以反馈的内容

  • 工具计算结果、转换结果或页面交互问题。
  • 隐私政策、广告 Cookie、第三方资源或数据处理相关问题。
  • 失效链接、乱码、重复内容、移动端显示异常。
  • 新工具建议、分类建议或文档改进建议。

回复说明

我们会优先处理影响访问、隐私、安全和核心工具可用性的问题。请在反馈中尽量提供页面地址、浏览器、设备类型和复现步骤，方便定位。
